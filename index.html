Warden

Rack Middleware for Authentication
  • Github
  • Wiki
  • Issues

Warden is an authentication solution for Ruby applications. Based on Rack, it's designed to work with any Rack based application, including Rails, Sinatra or any other Rack based framework. If a framework is not your thing, it works just great with raw Rack applications as well.

Warden provides a consistent API throughout Rack applications. This means that any component built for Rack can require an authenticated user in a known way, without having to worry about your applications authentication requirements. They might be password, open id, oauth or any other scheme modern web applications use. To Rack machinery though, the authentication is as simple as: 

env['warden'].authenticate!
With this one statement, any Middleware or application can request authentication customised to your applciation.

  • Rack Based
  • Consistent API
  • Transportable
  • Any Type of User
  • Multiple Users
  • Multiple User Scopes
